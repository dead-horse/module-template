## Installation

```bash
$ npm install module-template
```

## Usage

- add your user template
- init project

```bash
      mtpl init <dir>

    user: add or remove user template
      mtpl user add
      mtpl user ls|list
      mtpl user rm|remove [id]
      mtpl user edit [id]
      mtpl user default [id]
```
